快速上手

Quickstart

知微是一个部署在 EdgeOne Makers 上的四层系统：应用层（看板+官网）、数据层（Edge KV 选题库 + 回流 API）、智能层（Expert + Skills）、自动化层（每日/每周报告）。

ZhiWei is a four-layer system on EdgeOne Makers: app (dashboard + site), data (Edge KV store + ingestion API), intelligence (Expert + Skills), and automation (daily/weekly reports).

  1. 打开 实时看板，查看命中率、时段热度、关键词置信度。
  2. Open the Dashboard to see hit-rate, time heat and keyword confidence.
  3. 在评分区输入候选选题，获取命中评分 + 理由 + 建议时段。
  4. Enter a candidate topic to get a score + reasons + best time slot.
  5. 用「多平台改写」把一稿生成公众号 / 小红书 / 抖音版本。
  6. Use multi-platform rewrite to produce WeChat / RED / Douyin variants.

Expert & Skills（智能层）

Expert & Skills

知微的智能层由一个 WorkBuddy Expert 和三个 Skill 组成，可直接在 WorkBuddy 中调用。

The intelligence layer is one WorkBuddy Expert plus three Skills, callable directly inside WorkBuddy.

选题评分官 · Expert

Topic Scorer · Expert

基于历史数据为候选选题打分并解释理由。

Scores candidate topics from history and explains why.

选题雷达 · topic-radar

扫描历史命中分布，产出选题建议。

Scans hit distribution, outputs topic ideas.

多平台改写 · multi-platform-rewrite

一稿多平台风格适配改写。

Rewrites one draft into platform-specific styles.

命中率复盘 · hit-rate-review

周期性复盘表现，沉淀数据洞察。

Periodic performance review into insights.

API 参考

API reference

边缘函数 API，部署于 EdgeOne Makers。KV 审批中时自动降级到内置快照。

Edge function APIs on EdgeOne Makers. Falls back to embedded snapshot while KV is under review.

GET  /api/stats     # 命中率 / 时段热度 / 关键词置信度
POST /api/score     # body: { title } → 命中评分 + 理由 + 时段
GET  /api/posts     # 历史内容列表
POST /api/posts     # 回流一条内容表现数据
